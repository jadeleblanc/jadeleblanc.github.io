# Jade Leblanc - Web Developer Portfolio

A modern, responsive portfolio website showcasing my journey as a web development student. Built with clean HTML5, CSS3, and JavaScript, featuring smooth animations, interactive elements, and a mobile-first design.

## 🌟 Features

- **Responsive Design**: Optimized for all devices from mobile to desktop
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content
- **Portfolio Sections**: About, Skills, Projects, and Contact information
- **Mobile-First**: Hamburger navigation and mobile-optimized layouts
- **Performance Optimized**: Fast loading with efficient CSS and JavaScript

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Design**: CSS Grid, Flexbox, CSS Variables
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter)
- **Hosting**: GitHub Pages

## 📱 Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About Me**: Personal introduction and statistics
3. **Skills & Technologies**: Technical skills organized by category
4. **Featured Projects**: Showcase of development projects
5. **Contact**: Contact form and social media links
